# Seppmail-ExchangeOnline-FAQ

> Exchange Online, connector, rule and certificate guidance for SEPPmail deployments.

**Partner resource by Synedat for the SEPPmail ecosystem.**

## Why this repository exists

Practical Exchange Online FAQ, troubleshooting notes and PowerShell-oriented guidance for SEPPmail appliance and seppmail.cloud integrations.

This repository is structured for public consumption and easy discovery across topics such as SEPPmail, Exchange Online, Microsoft 365, Azure, API automation, PowerShell and operational runbooks.

## Included content

- `Enable OnPremConnectors.md`
- `Handling onmicrosoft.com domains.md`
- `Test Transportrules.md`
- `Create Certificates.md`

## Quick start

1. Review the FAQ topics in the root folder.
2. Validate connector and rule expectations against the official SEPPmail Exchange Online guide.
3. Use the examples as a starting point for tenant-specific operational documentation.
