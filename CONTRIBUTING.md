# Contributing

Thanks for helping improve this repository.

## What belongs here

Please keep contributions focused on:

- Exchange Online integration guidance for SEPPmail
- reproducible operational notes
- concise troubleshooting instructions
- corrections, readability improvements, and documentation structure
- references to official SEPPmail documentation

## What does not belong here

Please do not add:

- customer-specific secrets, screenshots, or tenant identifiers
- unsupported workaround collections without context
- unreviewed production code snippets that imply vendor support
- large architecture documents better hosted in private project repositories

## Writing guidelines

- Prefer short sections and action-oriented headings.
- Explain *why* an issue occurs before describing *how* to fix it.
- Use neutral, professional language.
- Link to official SEPPmail or Microsoft documentation where appropriate.
- Replace real customer data with placeholders such as `contoso.example` or `tenant.example`.

## Pull request checklist

Before opening a pull request, please confirm:

- the change is documentation-first and safe for public publication
- examples are anonymised
- links were checked
- the text is understandable for consultants and administrators
- the contribution does not contradict official vendor documentation
